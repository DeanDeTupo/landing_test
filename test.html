Что знаю о CRM и CMA? (ничего)

CRM

CRM(от англ. CustomerRelationshipManagment — «Система управления взаимоотношениями с клиентами») - система взаимодействия с пользователем, отслеживания его действий , хранения о нём данных, анализа этих данных. Всё это не всегда реальзовано в одной отдельной CRM-системе. Для разделения взаимодейсвия с клиентом, может быть реализовано несколько систем, каждая со своей функциональностью.

CMS

CMS(от англ. ContentManagmentService - "система управления контентом") - это система для написания сайтов. У неё есть движок для различных функций: для личных кабинетов, платежей, создания постов, комментариев и ид, а также возможность для настройки визуального отображения содержимого. Пользователь создаёт с помощью CMS свой сайт, добавляет необходимый функционал, настраивает дизайн, а дальше управляет содержимым сайта без написания строчек кода.
